person(1) heeft plural( NUM, fruit( 0 ) ) voor elke plural( DENOM, fruit( 1 ) ). Noteer de verhouding van fruit( 0 )s tot fruit( 1 ) als een vereenvoudigde breuk.

NUM / DENOM

Verhoudingen kunnen op verschillende manieren worden geschreven, maar betekenen hetzelfde.

Je kunt een verhouding schrijven als twee getallen, gescheiden door een dubbele punt.

NUM:DENOM

Je kunt het schrijven in een zin, zoals:

NUM \text{ staat tot } DENOM

Of je kunt de verhouding schrijven als een breuk.

\dfrac{NUM}{DENOM}=fractionReduce( NUM, DENOM )

De verhouding van fractionReduce( NUM, DENOM ) tot fruit( 0 ) geschreven als een vereenvoudigde breuk is fruit( 1 ).
