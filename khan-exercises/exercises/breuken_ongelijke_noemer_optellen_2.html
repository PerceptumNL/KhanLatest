Tel fraction(NUM1,DEN1) en fraction(NUM2,DEN2) bij elkaar op.

(NUM1/DEN1)+(NUM2/DEN2)

Om deze twee breuken bij elkaar op te kunnen tellen, maken we de noemers eerst gelijk.

Je kiest voor de nieuwe noemer een zo klein mogelijk getal. Dat getal moet deelbaar zijn door de beide oude noemers.
We zoeken dus het kleinste gemeenschappelijke veelvoud van DEN1 en DEN2.
Dit is het getal LCM, ofwel \color{#3DBA5B}{\text{het kgv van de noemers is LCM}}.

\text{Je kunt ook altijd de twee noemers met elkaar vermenigvuldigen.} \text{Je krijgt dan } \color{#3DBA5B}{DEN1} * \color{#3DBA5B}{DEN2} = \color{#3DBA5B}{DEN1*DEN2} \text{ voor de nieuwe noemer.} \text{Als je dit getal kiest, krijg je wel iets meer rekenwerk.}

De waarde van een breuk verandert niet als je de teller en de noemer van die breuk met hetzelfde getal vermenigvuldigt.
We gaan beide breuken zodanig opschrijven dat ze allebei de noemer LCM krijgen. \color{#f00}{\text {Vermenigvuldig de teller en de noemer van de eerste breuk met FAC1.}} \color{#553ED6}{\text {Vermenigvuldig de teller en de noemer van de tweede breuk met FAC2.}}
\color{#f00}{fraction(NUM1,DEN1)} + \color{#553ED6}{fraction(NUM2,DEN2)} \ \ \ = \ \ \ \ \color{#f00}{fraction(NUM1*FAC1 , LCM)} + \color{#553ED6}{fraction(NUM2*FAC2 , LCM)}

De noemers zijn nu gelijk, dan kun je de tellers bij elkaar optellen.

\color{#f00}{fraction(NUM1,DEN1)} + \color{#553ED6}{fraction(NUM2,DEN2)} \ \ \ = \ \ \ \ \color{#f00}{fraction(NUM1*FAC1 , LCM)} + \color{#553ED6}{fraction(NUM2*FAC2 , LCM)} \ \ \ = \ \ \ \ fraction(NUM1*FAC1 + NUM2*FAC2 , LCM)

Je kunt het antwoord nog vereenvoudigen.

\color{#f00}{fraction(NUM1,DEN1)} + \color{#553ED6}{fraction(NUM1,DEN2)} \ \ \ = \ \ \ \ \color{#f00}{fraction(NUM1*FAC1 , LCM)} + \color{#553ED6}{fraction(NUM2*FAC2 , LCM)} \ \ \ = \ \ \ \ fraction(NUM1*FAC1 + NUM2*FAC2 , LCM) \ \ \ = \ \ \ \ fraction(reduce(NUM1*FAC1 + NUM2*FAC2 , LCM)[0],reduce(NUM1*FAC1 + NUM2*FAC2 , LCM)[1])
