Help ook mee met iktel.nl.

  • Geef feedback voor iktel.nl via de feedback functie. Deze vind je in de rechter onderhoek van je browser.
  • Maak video's voor iktel.nl

      • Een filmpje maken is niet moeilijk, maar je moet wel de volgende benodigheden hebben:

    1. Een computer of tablet met internettoegang
    2. Een microfoon (vaak ingebouwd in een laptop)
    3. Een YouTube-account (login kan met je Google-account die je ook nodig hebt voor iktel.nl)

      4. We werken met de website www.screencast-o-matic.com. Zij bieden gratis software waarmee je filmpjes kunt maken van wat er op je beeldscherm gebeurt. Hierbij komt je eigen gezicht ook niet in beeld, alleen je stem en de aantekeningen in het tekenprogramma Paint worden opgenomen.

      Hieronder volgt een instructiefilmpje voor het maken van filmpjes voor iktel.nl

  • Ga naar wiki.iktel.nl en help ons het curriculum van oefeningen te verbeteren en aan te vullen.
  • Werk mee in de code om iktel.nl te verbeteren. Hiervoor gebruiken wij Github
  • Doe een iktel.nl pilot op jouw school! Stuur een mailtje naar info@iktel.nl
