De buurtsupermarkt moet weer een nieuwe bestelling plaatsen voor de groenteafdeling. Ze hebben nu start kratten met groente. Gemiddeld verkopen ze gemiddeld kratten met groente in een week.

Hoeveel kratten moet de buurtsupermarkt minstens bestellen?

Answer

We willen weten hoeveel kratten groente er "missen" (hoeveel er minder zijn dan nodig).

Dit is een aftreksom.

Je moet het aantal kratten groente aanvullen tot het gemiddeld aantal verkochte kratten.

Hiervoor trek je de start kratten die je al hebt van het aantal kratten af dat je in totaal moet hebben.

Het antwoord is dus gemiddeld - start = Answer.

randRange(10,99) randRange(10,99) randRange(2,50) start + erbij - end

In een bus zitten start mensen. Na de bushalte zitten er end mensen in de bus. Er zijn erbij mensen ingestapt.

Hoeveel mensen zijn er bij de halte uitgestapt?

Answer

Dit is een optel- en een aftreksom.

Als er niemand uitgestapt was, zaten er start + erbij = start + erbij mensen in de bus.

Er zitten nu end mensen in de bus. Dus er zijn start + erbij - end = Answer mensen uitgestapt.
