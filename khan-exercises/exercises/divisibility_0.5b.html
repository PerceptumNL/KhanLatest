Getallen die deelbaar zijn door B zijn veelvouden van B, namelijk B, B*2, B*3, B*4.....

Over het algemeen is elk getal dat geen restgetal oplevert bij delen door B een veelvoud van B.

We kunnen beginnen met elk van onze antwoordmogelijkheden te delen door B.

WRONG \div B = floor( WRONG / B )\text{ rest }( WRONG % B )

Het enige antwoordmogelijkheid dat geen restgetal heeft bij deling is het getal A, want \quadFACTOR * B = A.
