De grafiek heeft een top en heeft een U-vorm. Aan beide zijden loopt hij oneindig door.

Dat is de beschrijving van een parabool.

Hyperbool

Deze grafiek nadert de twee assen. Het is ook een U-vorm aan beide kanten.

plot( function( x ) { return (B/A) * x; }, [-10, 10], { stroke: "gray" } ); plot( function( x ) { return (B/A) * -1 * x; }, [-10, 10], { stroke: "gray" } );

Deze kromme is daarom een hyperbool.
