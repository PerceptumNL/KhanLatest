Welk getal is kleiner dan \color{#FFA500}{A}? Je kunt kiezen uit \color{#6698FF}{B} en \color{#6698FF}{C}.

Laten we eerst kijken naar hoeveel \color{#FFA500}{A} is:

init({ range: [ [0, 12], [-1, 1] ] }); drawCircles( A, "#FFA500" );

En nu hoeveel \color{#6698FF}{B} en \color{#6698FF}{C} zijn:

init({ range: [ [0, 12], [-1, 1] ] }); drawCircles( B, "#6698FF" );
init({ range: [ [0, 12], [-1, 1] ] }); drawCircles( C, "#6698FF" );
Nu kun je eigenlijk al zien welk getal groter is.

Voor de duidelijkheid zetten we de getallen op een getallenlijn:

init({ range: [ [LOWER_BOUND - 1, UPPER_BOUND + 1], [-1, 1] ] }); line( [ LOWER_BOUND, 0 ], [ UPPER_BOUND, 0 ] ); for ( var x = LOWER_BOUND; x <= UPPER_BOUND; x++ ) { line( [ x, -0.2 ], [ x, 0.2 ] ); } style({ stroke: "#000000", strokeWidth: 3.5 }); line( [ 0, -0.2], [0, 0.2]); label( [ 0, -0.53 ], "0", "center", { color: "#000000" }); style({ stroke: "#FFA500", fill: "#FFA500" }); graph.orangeDot = circle( [ A, 0 ], 0.10 ); style({ stroke: "#6698FF", fill: "#6698FF" }); graph.yellowDot = circle([B,0],0.10); style({stroke: "#6698FF", fill: "#6698FF"}); graph.blueDot = circle([C,0],0.10); label([A,-0.53], A, "center", {color: "#FFA500"}); label([B,-0.53], B, "center", {color: "#6698FF"}); label([C,-0.53], C, "center", {color: "#6698FF"});
Nu is duidelijk te zien dat \color{#6698FF}{B} kleiner is dan \color{#FFA500}{A}, en \color{#6698FF}{C} niet.
