Wat is de helft van Number?

Answer

Stel je hebt Number knikkers.
init({ range: [[-24, 24], [-5, 10]], scale: [20, 20] }); style({ stroke: "black", strokeWidth: 2 }); for(i=1;i <= Answer;i++) {circle([-25+(3*i), 0], 1, { fill: "teal", stroke: "gray", strokeWidth: 5, opacity: 0.6 }); } for(i=1;i <= Answer;i++) {circle([-25+(3*i), 5], 1, { fill: "teal", stroke: "gray", strokeWidth: 5, opacity: 0.6 }); }
Door de helft van de knikkers weg te halen hebben we er in totaal:
init({ range: [[-24, 24], [10, 15]], scale: [20, 20] }); style({ stroke: "black", strokeWidth: 2 }); for(i=1;i <= Answer;i++) {circle([-25+(3*i), 12], 1, { fill: "teal", stroke: "gray", strokeWidth: 5, opacity: 0.6 }); }
Door terug te tellen kom je uit op Answer knikkers.
