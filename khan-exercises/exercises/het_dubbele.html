Wat is het dubbele van Number?

Answer

Stel je hebt Number knikkers.
init({ range: [[-24, 24], [-5, 5]], scale: [20, 20] }); style({ stroke: "black", strokeWidth: 2 }); for(i=1;i <= Number;i++) {circle([-25+(3*i), 0], 1, { fill: "teal", stroke: "gray", strokeWidth: 5, opacity: 0.6 }); }
Daar komen bij het verdubbelen precies Number knikkers bij. Nu hebben wij er in totaal:
init({ range: [[-24, 24], [0, 10]], scale: [20, 20] }); style({ stroke: "black", strokeWidth: 2 }); for(i=1;i <= Number;i++) {circle([-25+(3*i), 3], 1, { fill: "teal", stroke: "gray", strokeWidth: 5, opacity: 0.6 }); } for(i=1;i <= Number;i++) {circle([-25+(3*i), 8], 1, { fill: "teal", stroke: "gray", strokeWidth: 5, opacity: 0.6 }); }
Door verder te tellen kom je uit op Answer knikkers.
