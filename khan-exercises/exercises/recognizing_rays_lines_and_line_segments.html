Dit een-dimensionale figuur heeft twee eindpunten en een gegeven lengte (en gaat niet oneindig door).

Een een-dimensionale figuur met twee eindpunten en een gegeven lengte is een lijnstuk.

line( END1, END2, { arrows: "->" } ); circle( END1, DOT_ATTRS ); circle( INNER2, DOT_ATTRS );
Halflijn

De figuur is een-dimensionaal en heeft een eindpunt. Het gaat echter oneindig door in de andere richting.

Therefore, this figure is a ray.

line( END1, END2, { arrows: "->" } ); line( END2, END1, { arrows: "->" } ); circle( INNER1, DOT_ATTRS ); circle( INNER2, DOT_ATTRS );
Lijn

De figuur is recht, een-dimensionaal, en gaat oneindig door in beide richtingen. Het heeft geen eindpunten.

Deze omschrijving betekent dat de figuur een lijn is.
