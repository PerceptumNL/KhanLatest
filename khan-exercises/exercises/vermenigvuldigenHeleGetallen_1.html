Stel we geven plural(A,C) elk plural(B,D), hoeveel plural(D) hebben alle plural(C) dan samen?

Dit kunnen we als volgt opschrijven:

init({ range: [ [ -1, 12 ], [ -3, B + 1 ] ], scale: [ 50, 50 ] }); KhanUtil.currentGraph.labelHolder = label( [ 0, B ], "\\Huge{\\color{#6495ED}{ A } \\times \\color{#28AE7B}{ B } = {?}}", "right" );
A * B
label( [ 0, B - 1 ], "\\text{We hebben " + plural( A, C ) + ".}", "right" ); drawRow( A, B - 3, "#6495ED", 1 );
label( [ 0, B - 2 ], "\\text{" + "Als er " + plural( A,C ) + ( A === 1 ? " is " : " zijn " ) + " met elk " + plural( B,D ) + ", hoeveel " + plural(D) + " zijn er dan in totaal?" + "}", "right" ); for ( var i = 1; i < B; i++ ) { drawRow( A, B - 3 - i, "#28AE7B", A * i + 1 ); }

In totaal zijn er plural(A,C) keer plural(B,D), dus A*B plural(D) (dit kun je controleren door de rijen te tellen).
