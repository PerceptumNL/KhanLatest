Door het aantal blokken van honderd te tellen, is te zien dat het getal dat het aantal honderdtallen weergeeft gelijk is aan A.

for ( var i = 0; i < A; i++ ) { label( [10.5 * i + 5, 5], "\\large{\\mathbf{100}}", { color: "black", background: "#a2bff4" } ); }

Omdat er geen blokken van tien zijn, is te zien dat het getal dat het aantal tientallen weergeeft gelijk is aan 0.

Door het aantal blokken van tien te tellen, is te zien dat het getal dat het aantal tientallen weergeeft gelijk is aan B.

for ( var i = 0; i < B; i++ ) { label( [5, -1.5 * i - 0.6], "\\large{\\mathbf{1\\:0}}", { color: "black" } ); }

Omdat er geen enkele blokjes zijn, is te zien dat het getal dat het aantal eenheden weergeeft gelijk is aan 0.

Door het aantal enkele blokjes bij elkaar op te tellen, is te zien dat het getal dat het aantal eenheden weergeeft gelijk is aan C.

for ( var i = 0; i < C; i++ ) { label( [1.5 * i + 0.5, 0.5], "\\mathbf{1}", { color: "black" } ); }

Het aantal blokjes is N.
