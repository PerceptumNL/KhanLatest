Wat is de uitkomst?

Answer

Dit is onderdeel van de tafel van B. Denk aan de tafel van B.

Dit is de tafel van B op volgorde:
1 x B = 1 * B
2 x B = 2 * B
3 x B = 3 * B
4 x B = 4 * B
5 x B = 5 * B
6 x B = 6 * B
7 x B = 7 * B
8 x B = 8 * B
9 x B = 9 * B
10 x B = 10 * B

Dus het antwoord is A x B = A * B.
